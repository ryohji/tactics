# バランスシミュレータ(rogue-27 S2 時点・2026-07-17)

`npm run balance -- --seeds 50 --out docs/balance/rogue27-s2-2026-07-17.jsonl` の結果。
シード 1..50 を greedy/cautious それぞれ1回ずつ(計100ラン)、`setTimeScaleForTest(0)` で
演出待ちを飛ばしてヘッドレス実行。実行時間 162.2秒。

rogue-27 S2 の変更点(ボットに影響するもの):
- 罠アイテム5種を廃止(罠は罠師「罠編み」のスキル化)。GADGETS が 7→2(砲塔・囮のみ)に
  縮小し、ガジェット枠の出土がすべて砲塔・囮になった。
- 初期パックから棘の罠が消えた(水薬1・投げナイフ2)。スキル0のボットは罠を一切使わない
  (旧方策の「15手ごとに罠設置」分岐は経路ごと消滅)。

|---|---|---|---|---|
| 死亡率 | 60.0% | 44.0% | 2.0% | 0.0% |

| 深度 p90 | 8 | 10 | 5 | 3 |

- 旧ベースラインは rogue-18 時点(層リセット・スキル体系・敵拡張の前)なので、差分は
  S2 単独の影響ではなく rogue-19b〜27 の累積(参考: rogue-19b 完了記録の同シード帯の
  貪欲死亡率は 70%)。
- greedy の死亡率 44% は罠(初期の棘の罠1つ)を失った不利より、rogue-21 以降の障壁・
  水薬プール拡充などの追い風が勝った形。深度分布(p50=5・p90=10)に極端な変化はなく、
  死因もグール・兵隊蟻が主因のまま(門番「茸の王」が新たに死因に入ったのは rogue-24 の
  敵拡張由来)。
- cautious は従来どおり手数上限で頭打ち(深度 p50=2)・死亡ほぼゼロの安全側で、
  方策間の弁別は維持されている。

## 一言

罠アイテム廃止(スキル0ボットの罠経路消滅+GADGETS 縮小)による難度の急変は見られない。
罠編み・結びの効き方はボットが罠スキルを装着しないため本計測には現れない — S3 以降で
ビルド特化方策(罠師など)を足したときに測る。
