# バランスシミュレータ ベースライン(rogue-18 時点・2026-07-11)

`npm run balance -- --seeds 200 --out docs/balance/baseline-2026-07-11-rogue18.jsonl` の結果。
シード 1..200 を greedy/cautious それぞれ1回ずつ(計400ラン)、`setTimeScaleForTest(0)` で
演出待ちを飛ばしてヘッドレス実行。実行時間 741.8秒(約12.4分)。

## cautious (n=200)

- 到達深度: min=0 p25=2 p50=2 p75=3 p90=5 max=10
- 平均ターン: 1884.4
- 死亡率: 2.5%
- 死因: グール×2, 岩グモ×2, 兵隊蟻×1

## greedy (n=200)

- 到達深度: min=0 p25=4 p50=7 p75=7 p90=8 max=10
- 平均ターン: 752.3
- 死亡率: 55.0%
- 死因: グール×64, 兵隊蟻×36, 岩グモ×8, 鬼火×1, 洞穴コウモリ×1

## 一言

cautious は明かりを絞って敵から退く分だけ生存率は高い(死亡率2.5%)がほとんど手数上限
(1500手/ラン)で頭打ちになり深度は伸びない(p50=2)。greedy は殴り合いで深く進む
(p50=7)が過半数(55%)がグール・兵隊蟻に倒される。両方策で明確に異なる分布が出ており、
シミュレータ自体は狙いどおり方策の違いを弁別できている。
