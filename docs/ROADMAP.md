## バックログ(各項目の詳細。順番は上の対応順を正とする)
- **バランスシミュレータ**: 実ゲームのストアを fake timers+演出待ちゼロで数千ラン回すヘッドレス統計(`npm run balance`, vite-node)。ボット方策(貪欲/慎重/ビルド特化)×シード集合で死亡深度分布・死因・種族別寄与・アイテム使用率を収集し、変更前後の A/B 差分で難度の意図せぬ変動を検出。ボットの操作列=行動ログでリプレイ検証エンジンと共用。詳細は DECISIONS 2026-07-11。
- **層による探索打ち切り/リセット**: 深度 N ごとに関門広間を置き、通過で上層(chambers/open/stubs)を崩落・破棄して現在広間を新しい根にする。メモリ・セーブ容量・描画コストを1層分で頭打ちにする狙い。帰還不能点として演出(関門前で準備を整える緊張)。層はスコア・バイオーム・門番ボスの単位にもなる。
- **スキル体系(マスタリー×スロット)**: マスタリー=永続メタ(使うことで育ち、ノードの選択肢を解禁。実績条件の離散式)、スロット=ラン内の力(初期2・関門+1・上限6目安、新規枠は3択ドラフト、組み替えは関門でのみ)。系統は 武技/盾/拳闘/隠密/罠師/灯火/甲殻 の7本・約30ノード、コスト1〜3。詳細は DECISIONS 2026-07-11「スキル体系の改訂」。
- **障壁システム**: HP(上限24固定)の上に 8/16/24 の上書き式シールド(スタックせず張り直しで上書き。下方上書きの判断が生まれるよう入手手段で強度を分ける: 消費財=8、仕掛け=16、深層仕掛け=24 目安)。自然減衰なし、層を跨ぐと消える。HP バー上に別色セグメント表示。
- **両手持ち・盾**: 右手/左手スロット。槍(リーチ2)・槌(防具無視・ノックバック)は両手、片手武器+盾で回避率+。スキルで「片手武器の両手保持=攻撃+」「両手武器の片手扱い=命中−で盾併用可」を解禁。命中/回避の乱数もシード列から引く(決定論の制約)。
- **共有スコアボード**: ゲーム側は送信のみ、記録と top100 保持は Cloudflare Worker(無料枠)が担う分担。送信スキーマ: スコア(最深・討伐・ターン)・シード・死因・ゲームバージョン(必須。Worker はバージョンごとに別ボードを保持)・スキル構成(スロット+マスタリー要約 — 上位の組み合わせが見える化しバランス計測を兼ねる)・将来は行動ログ。関門通過時のスナップショット送信(ラン ID で upsert、死亡で確定)により長いランも載る。発展形として行動ログのリプレイ検証(Worker→repository_dispatch→GitHub Actions で決定論モデルにより再シミュレート)。制約: ゲーム内乱数はすべてランのシード列から引くこと(Math.random 禁止)。リプレイ検証とゴールデンテストの前提。
- **ローカルスコアボード**: ラン履歴(最深・討伐・ターン・死因・シード・バージョン・日付)を localStorage の別キーで永続化しタイトル画面に表示。日付由来シードの「本日の迷宮」。
- **図鑑・実績**: 討伐図鑑(種別・初討伐深度)・アイテム図鑑(合成の最高品質)・実績(既存トレードオフを攻める形: 明かりを絞ったまま層を通過、罠だけで5討伐、無傷で関門到達など)。死んでも残るメタ要素だが、報酬は知識・展示とスキル選択肢のアンロックに限定。
- **層の門番(ミニボス)**: 関門に兵隊アリ級を1体。撃破でトロフィー+スキルポイントボーナス。層クリアの句読点。
- **遺物・持ち帰り(push-your-luck)**: 層ごとに1個の遺物を配置。関門で「降りる/脱出して収集を確定」を選べるようにして、無限降下に引き際の意思決定を入れる。展示棚をタイトル画面に。
- **敵ロスター拡張**: 新規7種(ネズミ群れ/毒ヘビ/酸粘体/胞子茸/骸骨術師=初の遠隔/サソリ/ミミック)+層バイオームの門番3種(Mushroom King/Giant/Yeti)。モデルは実在確認済み(Quaternius CC0・Poly by Google CC-BY・KayKit CC0)。「対抗手段が先、脅威が後」の配置原則、解毒の水薬新設、対抗スキル2ノード(掲盾・不浸の膜)、毒=障壁素通り/酸=障壁倍打の裁定、層帯再配置と深度係数を含む。詳細は DECISIONS 2026-07-11「敵ロスター拡張」。
- **小粒の先送り(過去イテレーションから)**: 敵種ごとの造形強化 / 環境音の深度変化 / スクリーンショットの README 掲載 / バランス調整(出土率・持続・威力)/ 状態異常のプレイヤー側適用 / ミニマップ / 経路プレビュー / バブルの画面端インジケータ化 / 通路に居る間の現在地ハイライト / 通路の繋ぎ直し / 本格FOV / シェル差分更新・チャンク化 / 徘徊AI
