# ビジョン

## 体験の核
**3D地形を3Dヘックス(FCC格子 / 菱形十二面体)で分割し、その上での移動・表示が気持ちよくハマっていること。**
プレイヤー(まずは作者自身)が、地形と盤面の重ね合わせを見て「これは破綻なく成立している」と腑に落ちる感覚。

## 世界観(2026-07-04 転換 → 2026-07-11 更新)
- 舞台: **蟻の巣状の 3D 迷宮**。無限の土の塊から、たいまつの明かりが届いた空洞だけがくり抜かれていく。ダンジョンは進行に合わせて生成され、下へ下へ枝分かれする。
- プレイ: たいまつひとつのエンドレス潜行。**明かりを広げれば癒えるが、目立つ** — このトレードオフが芯。どこまで深く行けるかがスコア。
- 3D性の含意: 12近傍(水平6+上3+下3)の立体移動、上下に敏感な敵の気づき、層をまたぐ縄張り — 巣の立体構造で FCC が本領を発揮する。

## なぜ面白いか / 差別化
2次元ヘックスの強み(隣接距離の均一・等方的な移動と射程)を、立方格子の対角問題を回避しながら3次元へ持ち上げている。
水平はヘックス、高さは層で読むという人間の直観に乗せられる。ここに**真の3D地形を自然に合成**できれば、高さ・張り出し・遮蔽を含む3D戦術の新しい土台になる。
