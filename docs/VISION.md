# ビジョン

## 体験の核
**3D地形を3Dヘックス(FCC格子 / 菱形十二面体)で分割し、その上での移動・表示が気持ちよくハマっていること。**
プレイヤー(まずは作者自身)が、地形と盤面の重ね合わせを見て「これは破綻なく成立している」と腑に落ちる感覚。

## 世界観・ユニット(2026-06-29 具体化)
- 舞台: **廃墟化した大聖堂**。壁の一部が崩れ、内部が外へ露出している。崩れた柱・落ちた天井・瓦礫。
- スケール: **人間サイズのキャラクター ≈ 1セル**。大聖堂は身廊の高い天井を持つ**壮大**な規模。
- ユニット: **飛行できる翼人・魔女**などのファンタジー設定。
- 飛行の含意: **室内外の3D空間ほぼ全体が移動可能領域**になる。床の上だけでなく高所・崩落部の隙間・聖堂の外までが戦場 → FCC の3D立体性(水平6+上3+下3)が本領を発揮する。地形は「飛び越え・回り込み・遮蔽を生む実体 occluder」として効く。

## ターゲット
まず自分 / 少数の実験。動くものを早く出し、仕様の妥当性を体で確かめることを優先する。

## なぜ面白いか / 差別化
2次元ヘックスの強み(隣接距離の均一・等方的な移動と射程)を、立方格子の対角問題を回避しながら3次元へ持ち上げている。
水平はヘックス、高さは層で読むという人間の直観に乗せられる。ここに**真の3D地形を自然に合成**できれば、高さ・張り出し・遮蔽を含む3D戦術の新しい土台になる。

## 北極星(North Star)
迷ったら問う: **「3D地形のヘックス分割・移動・表示が、気持ちよくハマっているか?」**
見栄えの豪華さでも機能の多さでもなく、この一点の心地よさを最優先する。
